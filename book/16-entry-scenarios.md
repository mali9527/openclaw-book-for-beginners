# 入门级开箱即用场景：5个你今天就能用起来

OpenClaw 装好之后，能做点什么有意思的事情？这一章给你介绍 5 个简单实用的入门场景，你今天就能配置起来用。

## 1. 每日晨间智能简报

这是最受欢迎的玩法，没有之一。

**它能做什么：**
每天早上固定时间，自动给你发简报，包含：
- 今天天气怎么样
- 你今天日程安排是什么
- 最近有什么新闻头条
- 你昨天有什么邮件需要处理

**你需要：**
- 安装 `weather` 技能
- 安装 `feishu-calendar` 技能（如果你用飞书日历）
- 配置一个定时任务（Heartbeat）

配置好之后，**你每天起床打开飞书就能看到简报**，不用自己一个个查，太方便了。

## 2. 邮件自动分类清理

你邮箱是不是几百上千封未读，很乱？让 OpenClaw 帮你整理。

**它能做什么：**
- 自动扫描你所有未读邮件
- 分类：垃圾邮件/广告/重要邮件/订阅邮件
- 垃圾邮件直接帮你退订
- 重要邮件标出来，给你写好回复草稿

**你需要：**
- 邮件账号支持 IMAP 访问（大部分邮箱都支持）
- 配置好邮件权限

用过一次你就会爱上，**帮你省出好多时间**。

## 3. 下载文件夹自动分类归档

你是不是下载了一堆文件，下载文件夹越来越乱，找东西找不到？

**它能做什么：**
自动扫描你的下载文件夹，按照文件类型分类：
- PDF 放到 `PDF/` 文件夹
- 图片放到 `Images/` 文件夹
- 安装包放到 `Installers/` 文件夹
- 压缩包放到 `Archives/` 文件夹

整理完告诉你整理了多少个文件，清爽了。

**你需要：**
- 不用额外技能，OpenClaw 自带文件操作工具

设置好，每周自动跑一次，下载文件夹永远干净。

## 4. 网页文章自动总结

看到一篇长文章，不想慢慢读，让 OpenClaw 帮你总结。

**它能做什么：**
你发给它一个网页链接，它自动：
- 提取文章正文内容
- 总结出核心要点，几千字变成几百字
- 你几分钟就能看完一篇长文

**你需要：**
- 安装 `tavily-search` 技能（自带网页提取）

真的太省时间了，我现在看长文都先让 AI 总结一下。

## 5. 个人 CRM，记住你的联系人

认识了新朋友，转头就忘记人家在哪认识的，做什么的？让 OpenClaw 帮你记。

**它能做什么：**
- 自动从邮件、日历、聊天记录提取联系人信息
- 帮你整理：姓名、邮箱、在哪里认识的、对方做什么的
- 你要找的时候，直接问它"我那个做产品经理的张三联系方式是什么"

**你需要：**
- 飞书已经有联系人信息的话，直接就能用
- 可以存在飞书多维表格里

时间长了，你的人脉网络就是你的宝贵财富，AI 帮你记住。

## 小结

这五个场景都是**开箱即用，配置好了就能用**，不需要你懂开发，适合入门试试水：

1. 晨间简报 → 每天早上给你汇总信息
2. 邮件清理 → 帮你整理邮箱退订垃圾
3. 下载分类 → 保持下载文件夹干净
4. 文章总结 → 长文变短，节省时间
5. 个人 CRM → 帮你记住联系人
