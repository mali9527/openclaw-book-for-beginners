# 入门级开箱即用场景：5个你今天就能用起来

OpenClaw 装好啦，技能也装好了，能做点什么有意思的呢？这一章给你介绍 5 个简单实用的入门场景，你今天就能配置起来用。

## 1. 每日晨间智能简报

这个绝对是最受欢迎的玩法，没有之一。

**它能做什么：**
每天早上固定时间，自动给你发简报，包含：
- 今天天气怎么样
- 你今天日程安排是什么
- 最近有什么热点新闻
- 你有什么待办任务

你起床打开飞书就能看，不用你一个个打开 APP 查，太方便了。

**你需要装这些技能：**
- `weather` - 查天气
- `feishu-calendar` - 读飞书日程
- `feishu-task` - 读飞书任务
- `tavily-search` - 搜新闻

配置一个定时任务（cronjob），每天早上 7 点自动运行，就搞定了。

**推荐指数：** ⭐⭐⭐⭐⭐ 谁用谁知道，用过都说好。

## 2. 邮件自动分类清理

你邮箱是不是几百上千封未读，乱得自己都不想看？让 OpenClaw 帮你清理。

**它能做什么：**
- 自动扫描你所有未读邮件
- 分类：垃圾邮件/广告邮件/重要邮件
- 垃圾邮件直接帮你退订
- 重要邮件标出来给你，还能帮你起草好回复

**你需要：**
- 邮箱开通 IMAP 访问（大部分邮箱都支持）
- 配置一下就能用

完了你的邮箱一下子就清爽了，省你好多时间。

**推荐指数：** ⭐⭐⭐⭐ 邮箱满了一定要试。

## 3. 下载文件夹自动分类归档

是不是你的下载文件夹一下载就满了，各种安装包图片压缩包乱得找不到东西？

**它能做什么：**
- 自动扫描你的下载文件夹
- 按文件类型分类：PDF 放 PDF 文件夹，图片放图片文件夹，安装包放安装包文件夹...
- 整理完告诉你一共整理了多少个文件

**你需要：**
- 不用额外技能，OpenClaw 自带文件操作能力
- 设置个定时任务，每周自动整理一次

从此你的下载文件夹永远干干净净，不用你手动拖来拖去。

**推荐指数：** ⭐⭐⭐⭐⭐ 谁用谁知道，一下子就清爽了。

## 4. 网页文章自动总结

看到一篇长长的文章，没时间慢慢读，让 OpenClaw 帮你总结。

**它能做什么：**
你把链接发给它，它自动：
- 提取文章正文内容
- 总结核心要点，几千字变成几百字
- 你几分钟就能看完一篇长文

**你需要：**
- `tavily-search` 已经装了就能用

真的太省时间了，我现在看长文都先让它总结一遍。

**推荐指数：** ⭐⭐⭐⭐⭐ 每天都用得到。

## 5. 个人 CRM 自动维护人脉

认识了新朋友，转头就忘记人家是做什么的，在哪里认识的，有没有这种情况？

**它能做什么：**
- 自动从邮件、聊天、日历里提取联系人信息
- 帮你整理好了：姓名、什么时候认识的、做什么的、联系方式
- 你要找的时候，直接问它"我那个做产品的张三联系方式是什么"，它找出来给你

**你需要：**
- 如果用飞书，可以存在飞书多维表格里
- 不用自己手动输，它自动提取

人脉是慢慢积累的，时间长了你就有一个自己专有的人脉库，很方便。

**推荐指数：** ⭐⭐⭐ 经常社交交朋友推荐一定要试。
