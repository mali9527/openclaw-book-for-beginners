# ClawHub 技能商店：怎么找、怎么装、怎么选

ClawHub 就是 OpenClaw 的官方技能商店，类似于手机上的应用商店，你需要什么功能搜一下，一条命令就能装上，不用自己写代码，特别方便。这一章我告诉你怎么用。

## 第一步：安装 clawhub 命令行工具

如果你已经装过了，可以跳过这一步。要是没装，一条命令就能装上：

```bash
npm install -g clawhub
```

装完验证一下：

```bash
clawhub --version
```

能输出版本号就是装好了。

## 搜索技能：怎么找到我想要的技能

搜索也特别简单，就是一条命令：

```bash
find-skills 你的关键词
```

其实 `find-skills` 就是包装了 `clawhub search`，你用哪个都一样。

比如你想要找 PDF 相关的技能，就输：

```bash
find-skills pdf
```

它就会给你列出所有匹配的技能，还有社区评分，评分越高越推荐。

## 安装技能：找到之后一条命令安装

搜到了你想要的技能，记下它的 slug 就是技能名字，然后：

```bash
find-skills install 技能名字
```

它就自动下载安装到你当前工作区的 `skills/` 目录下面了，就能用了，特别方便。

> 小知识：技能搜索顺序是：当前项目 skills → 本地 ~/.openclaw/skills → 内置技能，所以你可以给不同项目装不同技能，很灵活。

## 安装完了怎么用？

一般技能装好之后，技能里面有个 `SKILL.md`，打开看看，它会告诉你：

- 需要配置什么环境变量（比如 API Key）
- 怎么调用，有什么注意事项

你按照它说的，把环境变量配置好，重启一下 OpenClaw 就能用了。

## 怎么选技能？看评分选高分的

ClawHub 上面的技能有社区评分，**你就选评分最高的那个就行**，不会错。评分是社区用户用出来的，高分肯定比较好用。

要是同一个功能有好几个技能，选评分最高的，没错。

## 我装了不需要了，怎么删除？

也很简单，直接删掉 `skills/` 下面对应的技能文件夹就行了，没有复杂的命令，就是这么简单。

## 小结

- ClawHub 就是 OpenClaw 的应用商店
- `find-skills 关键词` 搜索技能
- `find-skills install 技能名` 安装技能
- 选评分最高的，不会错
- 用完删掉直接删文件夹就行
