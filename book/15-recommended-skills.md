## 第一类：核心三件套（强烈建议都装上）

这三个技能几乎谁都用得上，装了就能让你的 OpenClaw 能力翻倍。

### 1️⃣ tavily-search —— 让 AI 能上网搜东西

大模型有个天生的毛病：它的知识是有截止日期的，今天发生了什么新闻它不知道。tavily-search 就是解决这个问题的，装了它，你的 AI 就能实时搜索互联网。

**你可以对它说：**
- "帮我搜一下今天有什么重要新闻"
- "搜一下 XX 公司最近有什么动态"
- "帮我查一下上海明天天气"

**为什么推荐它：**
- 专门为 AI Agent 优化的搜索引擎，返回的结果干净准确，不是一堆广告
- 每个月有 1000 次免费额度，普通人完全够用
- 是 ClawHub 上下载量最高的技能之一，社区验证过，靠谱

**推荐指数：** ⭐⭐⭐⭐⭐ 几乎人人都要装

**安装命令：**
```bash
openclaw skills add tavily-search
```

**推荐指数：** ⭐⭐⭐⭐⭐ 每天都用得到

**安装命令：**
```bash
openclaw skills add summarize
```

## ⚠️ 安全提示：装技能之前，多留个心眼

ClawHub 上面的技能都是社区用户贡献的，大多数都很好用，但是安全意识还是要有：

1. **看下载量和评分**：下载量大、评分高的技能，一般都靠谱
2. **看作者**：知名作者或者官方认证的更放心
3. **不确定就看看源码**：技能就是一个文件夹，你点进去看看代码，有没有奇怪的东西
4. **敏感技能要谨慎**：涉及你的邮箱密码、银行信息这些的技能，一定要仔细看看再装
5. **建议用 Docker 隔离运行**：如果你不确定一个技能安不安全，可以在 Docker 容器里跑 OpenClaw，这样即使出问题也不会影响你的电脑

记住一句话：**技能就像手机 APP，大品牌大厂的放心装，不认识的先看看评价再说。**
