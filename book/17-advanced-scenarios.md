# 进阶级玩法：三个让你效率翻倍的进阶场景

入门场景玩过了，我们来看三个更有意思的进阶玩法，帮你效率再翻倍。

## 1. 打造你的"第二大脑"，随时存随时搜

你肯定有这个经历：看到一篇好文章，一个好点子，当时存起来了，后来要用怎么也找不到。

用 OpenClaw 打造个人第二大脑，解决这个问题。

**它能做什么：**
- 你随时给它发一段文字、一个链接、一张图片
- 它自动帮你整理分类，提取关键词，存在向量数据库里
- 你后来想找，直接问它"我之前存过一篇关于OpenClaw的文章"，它就能找出来给你

**你需要：**
- OpenClaw 原生支持 RAG，不用额外装技能就能用
- 你只要把文件放到你的工作区，它就能索引

真正做到"**你负责想到，它负责找到**"。

## 2. 手机上管理 GitHub，随时处理 PR

你出门在外，GitHub 有个 PR 需要 review，需要合并，电脑不在身边，怎么办？

**它能做什么：**
- 你在 Telegram/飞书 直接让 OpenClaw 帮你：
  - "看看这个 PR 改了什么"
  - "review 一下这个 PR 有没有问题"
  - "没问题，合并它"
- 它就能帮你做完，不用开电脑

**你需要：**
- 安装 `github-cli` 技能
- 配置 GitHub 权限

对于经常需要看 PR 的开发者来说，太方便了，出门也能处理工作。

## 3. 多个 AI 协作，一起写一本书

你现在看的这本书，就是**多个 AI 协作写出来的**：

- 一个做搜索收集资料
- 一个写大纲
- 一个写内容
- 最后一个检查校对

当然，最后还是要人审核，但是大部分工作 AI 都做了。

**它能做什么：**
你可以搭建自己的小团队：
- 研究员：收集资料整理信息
- 作者：写内容
- 编辑：改稿子检查错别字
- 项目经理：管进度排计划

一群 AI 帮你干活，你当老板审稿就行了。

**我们这本书就是这么做出来的：**
- 人类定大纲定方向
- AI 写初稿
- 人类审核修改
- 最后 AI 导出 PDF

比人类自己从头写快太多了，而且质量也不差。

这就是未来的写作方式：**人类把控方向，AI 出力**。

## 小结

入门场景帮你解决日常重复小事，进阶场景帮你解决更大的问题：

1. 第二大脑 → 帮你记住所有东西，找得到
2. GitHub 手机管理 → 出门也能处理工作
3. 多 AI 协作写书 → 人类指挥，AI 干活
