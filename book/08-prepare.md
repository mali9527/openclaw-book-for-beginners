# 准备工作：你开始之前需要准备这些东西

好了，朋友，前面我们把原理都说清楚了，接下来就要动手安装 OpenClaw 了。在开始之前，我们看看你需要准备哪些东西——别怕，东西真的不多，也不难。

## 第一，你需要 Node.js 22 以上版本

OpenClaw 是用 TypeScript 写的，运行需要 Node.js 环境，所以你得先装一个 Node.js，版本要 22 或者更高。

### 怎么看你有没有安装，版本对不对？

很简单，你打开终端（就是 macOS 里的 Terminal 这个程序），输入下面这行命令，回车：

```bash
node --version
```

如果输出像是 `v22.0.0` 这样，版本号大于等于 22，那就没问题，可以用。

如果提示你 `command not found`，那就是没安装，或者安装了没放到 PATH 里，需要重新装一下。

### 怎么安装 Node.js 呢？

两种方法，你选一个方便的就行：

#### 方法一：图形界面安装（适合新手朋友）

1. 打开 Node.js 官网：https://nodejs.org/
2. 下载那个 **LTS 版本**——就是长期支持版，稳定，适合我们用
3. 双击你下载的安装包，跟着提示一步步点"继续"→"同意"→"安装"，输入你电脑密码就好了
4. 安装完了把终端关掉重新打开，再运行 `node --version`，能看到版本号就是成功了

#### 方法二：用 Homebrew 安装（适合 macOS 用户，你用过 Homebrew 就选这个）

打开终端，直接输：

```bash
brew install node
```

然后等它安装完就好了，不用管它。

> 💡 小提示：安装完 Node.js，npm 也一起给你装好了，不用单独装。

## 第二，你需要一个大模型的 API Key

OpenClaw 它就是一个平台，对不对？它自己不会"思考"，思考需要大模型，所以你得有一个大模型的 API Key 才能用。

### 我该选哪个大模型呢？

我给你整理一个表格，你自己对着选：

| 厂商 | 优点 | 缺点 | 适合谁 |
|------|------|------|--------|
| OpenAI GPT-4o | 能力最强，出错少 | 贵一点，需要科学上网 | 不在乎钱，想要最好效果 |
| Anthropic Claude 3/Opus | 长文本处理好，不容易被提示注入 | 同样偏贵，需要科学上网 | 想要高质量，处理长文档 |
| 百度文心一言 | 国内直接能用 |  | 国内朋友不想折腾科学上网 |
| 阿里通义千问 | 国内直接能用 |  | 国内朋友 |
| DeepSeek | 性价比高 |  | 想要性价比 |

你刚开始玩，选哪个都行，OpenClaw 支持所有主流大模型，**以后想换随时能换**，不用重新安装，改一下配置就好了，放心。

### 怎么拿到 API Key 呢？

其实不管哪个厂商，流程都差不多：

1. 去厂商的官方网站注册账号
2. 充值一点钱——真不用多，几块钱几十块钱够你玩很久了
3. 进到 API 管理页面，创建一个新的 API Key
4. 复制出来找个地方存好，后面配置 OpenClaw 要用

> ⚠️ 重要提醒：你的 API Key 一定一定保存好，别告诉别人，也别放到 GitHub 公开仓库里——别人拿到了就能用你的钱消费。

## 第三，你需要会一点基础的电脑操作

你不需要会写代码，真的，但是你需要会这几件事：

- ✅ 会打开终端
- ✅ 能跟着教程复制粘贴命令到终端运行
- ✅ 出了错，能看懂错误提示，复制下来去搜索

就够了。

如果你说"我连终端都没开过"，那也没问题，你打开百度搜一下"怎么打开终端 macOS"，或者"怎么打开 CMD Windows"，两分钟就学会了。

## 还要准备什么别的吗？

真的不用了，**就这么三样东西**：

1. Node.js 22 以上
2. 一个大模型的 API Key
3  会一点基础操作

你看，是不是很简单？
