# 准备工作：你开始之前需要准备这些东西

好了，原理讲得差不多了，接下来我们要动手安装 OpenClaw 了。在开始之前，我们看看你需要准备哪些东西，别怕，东西不多，也不难。

## 第一，你需要 Node.js 22 以上版本

OpenClaw 是用 TypeScript 写的，运行需要 Node.js 环境。你需要安装 Node.js 22 或者更高版本。

### 怎么看我有没有安装？是什么版本？

打开终端（Terminal），输入下面这个命令，回车：

```bash
node --version
```

如果输出类似 `v22.0.0` 这样的，版本号大于等于 22，就可以用。

如果提示"command not found"，就是没安装，或者没放到 PATH 里。

### 怎么安装 Node.js？

两种方法都可以，选一个你喜欢的：

#### 方法一：图形界面安装（适合新手）

1. 打开 Node.js 官网：https://nodejs.org/
2. 下载 **LTS 版本**（长期支持版，稳定）
3. 双击下载的安装包，跟着提示点"继续"→"同意"→"安装"，输入电脑密码就可以了
4. 安装完关闭终端再重新打开，运行 `node --version` 能看到版本号就是成功了

#### 方法二：Homebrew 安装（适合 macOS 用户，用过 Homebrew 的）

打开终端，输入：

```bash
brew install node
```

等待安装完成就可以了。

> 小提示：安装完之后，node 和 npm 都会给你装好，不用单独装 npm。

## 第二，你需要一个大模型 API Key

OpenClaw 只是一个平台，它自己不会"思考"，思考需要大模型，所以你需要一个大模型的 API Key。

### 我该选哪个大模型？

看你的情况：

| 厂商 | 优势 | 劣势 | 适合谁 |
|------|------|------|--------|
| OpenAI GPT-4o | 能力最强，不会错 | 贵一点，需要科学上网 | 不在乎钱，想要最好效果 |
| Anthropic Claude 3/Opus | 长文本处理好，不容易被注入 | 同样贵，需要科学上网 | 想要高质量，处理长文档 |
| 百度文心一言 | 国内可直接用 |  | 国内用户不想科学上网 |
| 阿里通义千问 | 国内可直接用 |  | 国内用户 |
| DeepSeek | 性价比高 |  | 想要性价比 |

你刚开始玩，选哪个都行，OpenClaw 支持所有主流大模型，后面想换随时可以换配置，不用重新安装。

### 怎么拿到 API Key？

不管哪个厂商，基本都是这个流程：

1. 去厂商官网注册账号
2. 充值一点钱（几块钱几十块钱够你玩很久了）
3. 去 API 管理页面，创建一个新的 API Key
4. 复制保存好这个 Key，后面配置 OpenClaw 要用

> 🔔 注意：你的 API Key 一定要保存好，不要告诉别人，不要放到 GitHub 公开仓库，别人拿到了会用你的配额花钱。

## 第三，你需要会一点基础的电脑操作

你不需要会写代码，但是你需要：

- ✅ 会打开终端
- ✅ 会跟着教程复制粘贴命令运行
- ✅ 能看懂终端提示的错误，实在看不懂能复制去搜

就够了。

如果你连终端都没开过，也没关系，网上搜一下"怎么打开终端 macOS/Windows"，两分钟就学会了。

## 还要准备什么吗？

其实不用了，就这三样：

1. Node.js 22+
2. 大模型 API Key
3. 基础电脑操作

就可以开始了。

## 我都准备好了，下一步做什么？

准备好了，我们下一章开始安装 OpenClaw，一步一步来，你跟着做就可以了。
