# 第一步：安装 OpenClaw - 跟着官方走准没错

好了，准备工作做好了，我们现在开始安装 OpenClaw。

我之前说过，OpenClaw 更新很快，如果我这里写的具体命令和官方不一样了，**请一定以官方为准**。所以我会告诉你去哪里找最新的官方安装命令，怎么验证安装成功，方法对了，命令变了也没关系。

## 安装命令其实很简单

OpenClaw 推荐用 npm 全局安装，打开终端，输入一行命令就可以了：

```bash
npm install -g openclaw@latest
```

输入完回车，然后等几分钟就安装好了。中间会输出一堆 warning，那些都不用管，正常的，只要最后没有说"错误"就没问题。

## 怎么验证安装成功了？

安装完之后，输入这个命令：

```bash
openclaw --version
```

如果输出了类似这样的版本号：

```
openclaw 1.0.0
```

那就说明安装成功了。

如果提示"openclaw: command not found"，那就是没安装成功，或者安装目录没放到 PATH 里。常见解决方法：

- 如果你用的是 macOS，试试重启终端再看
- 如果还是不行，搜索"npm global bin path"，把 npm 的 bin 目录加到你的 PATH 环境变量里

## 初始化配置

安装成功了，接下来运行初始化向导：

```bash
openclaw onboard --install-daemon
```

这个命令会帮你做初始化配置，跟着向导一步步走就行了，我这里把每一步说清楚：

### 第一步：安全提示

它会先给你一个安全提示，问你"你认为当前环境足够安全吗（可以接受被攻击破坏）"，如果你是在你自己的个人电脑上，选"Yes"就可以了。

### 第二步：选择配置模式

向导会问你配置模式，新手推荐选择 **QuickStart**，先快速跑起来，所有配置后面都能改，没问题。

### 第三步：选择模型厂商

接下来会让你选择模型厂商，如果你不差钱，想要最好效果，直接选 **Anthropic** 或者 **OpenAI**。如果想要性价比，选国内的厂商（百度文心、阿里通义、DeepSeek 都有）。

如果你想要的厂商不在列表里，选 **Skip for now**，后面我们自己改配置文件，我后面会教你怎么改。

### 第四步：配置 API Key

选好厂商，它会让你输入 API Key，把你之前准备好的 API Key 粘贴进去就行。

### 第五步：选择具体模型

厂商选好 API Key 输完，接下来选具体模型，比如你选了 OpenAI，它会让你选 `gpt-4o` 还是 `gpt-3.5-turbo`，选你想用的就行。

如果你想要自己输入模型 ID，选 **Enter model manually** 自己输入。

### 第六步：配置消息渠道

接下来问你要不要配置 Channel，也就是你想在哪里和 OpenClaw 聊天。国内用户推荐先用 **Skip**，我们后面专门一章讲怎么配置飞书，先让基础的跑起来。

### 第七步：安装技能

接下来问你要不要现在安装技能，同样，新手推荐 **Skip**，先跑通基础流程，后面装技能很简单，我们后面专门讲。

### 第八步：配置 Hooks

最后会让你选 Hooks，推荐你空格选中这两个：

- `command-logger`：把所有命令相关事件记录到审计文件，排错非常有用
- `session-memory`：`/new` / `/reset` 的时候保存会话上下文，避免状态丢了

选中这两个就可以了，回车继续。

然后它就会自动完成配置，最后会给你一个 WebUI 地址，类似这样：

```
WebUI: http://127.0.0.1:18789/token/xxxxxx
```

## 验证安装成功

你复制这个地址，在浏览器打开，如果能看到 OpenClaw 控制台界面，能和它对话，那就说明安装成功了！

如果你配置了 API Key，现在就能在 WebUI 和它聊天了，试试问它一个问题，看能不能正常回复。

## 如果安装出错了怎么办？

- 首先看错误提示说什么，复制错误提示去 GitHub 搜一搜，很多人都碰到过同样问题
- 如果搜不到，去 OpenClaw GitHub 开一个 Issue，社区有人帮你

## 小结

- 安装就一行命令：`npm install -g openclaw@latest`
- 初始化：`openclaw onboard --install-daemon`，跟着向导走就行
- 新手选 QuickStart，先跑起来，后面配置都能改
- 最后能打开 WebUI 能对话，就是成功了
