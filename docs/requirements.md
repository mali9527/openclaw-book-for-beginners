# 需求说明书：OpenClaw 小龙虾入门指南

## 项目定位

一本面向**零基础普通人**的 OpenClaw 入门电子书，帮助读者从"不知道什么是 AI 智能体"到"能独立搭建并使用自己的 AI 助理"。

## 目标读者

- 对 AI 感兴趣但没有技术背景的普通人
- 会基本的电脑操作（能打开终端、复制粘贴命令）
- 不需要会写代码

**不适合**：已经在用 OpenClaw 的开发者，想深入源码级分析的人。

## 内容范围

### 第一部分：AI 智能体基础（概念科普）
让读者先懂原理再动手，理解 AI 智能体是什么、五大核心组件、为什么现在火了。

### 第二部分：认识 OpenClaw（产品介绍）
OpenClaw 是什么、从哪来、架构、记忆系统——建立整体认知。

### 第三部分：安装配置上手（实操教程）
准备工作 → 安装 → 配置模型 → 接入飞书 → 设定性格——一步步跟着走。

### 第四部分：技能系统（核心能力）
什么是技能、ClawHub 怎么用、推荐技能——OpenClaw 的灵魂。

### 第五部分：实际场景和玩法（价值落地）
入门场景、创意案例、进阶玩法、完整项目实战——让读者真正用起来。

### 第六部分：排错和安全（保驾护航）
常见问题排查、安全最佳实践——出了问题知道怎么办。

## 风格规范

| 规范 | 要求 |
|------|------|
| 语气 | 轻松口语化，像朋友聊天，不是教科书 |
| 术语 | 每个技术术语必须用大白话解释一遍 |
| 类比 | 多用生活化的比喻（菜谱、电话总机、笔记本等） |
| 结构 | 多用小标题、表格、列表，方便快速浏览 |
| 长度 | 每章 2000-4000 字，不啰嗦也不太短 |
| 💡 提示框 | 用 `> 💡` 解释容易困惑的概念 |

## 输出格式

| 格式 | 说明 | 状态 |
|------|------|------|
| Markdown 源文件 | GitHub 在线阅读 | ✅ 已有 |
| A4 PDF | 电脑端阅读 / 打印 | ✅ 已有 |
| 手机 PDF (9×16cm) | 手机端阅读 | ✅ 已有 |
| HTML 电子书 | 未来计划 | 🔜 待做 |

## 构建方式

- 章节源文件在 `src/chapters/`
- 构建脚本 `scripts/build.sh` 一键合并 + 生成 PDF
- 模板配置在 `templates/` 下按输出格式分目录
